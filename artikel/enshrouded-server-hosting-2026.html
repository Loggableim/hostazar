Enshrouded Server – Hosting-Guide & Setup für Koop 2026

Enshrouded hat die Survival-Szene im Sturm erobert. In diesem Guide erfährst du, wie du deinen eigenen Enshrouded Dedicated Server aufsetzt – ob günstig gemietet oder auf deinem eigenen Rechner installiert.

Was ist Enshrouded?

Enshrouded ist ein actionreiches Survival-RPG mit Basisbau, Crafting und einer riesigen offenen Welt. Das Besondere: Der titelgebende "Shroud" – ein tödlicher Nebel, der die Welt überzogen hat. Anders als viele andere Survival-Spiele setzt Enshrouded stark auf eine Story-Kampagne und erkundbare Dungeons. Im Koop-Modus mit bis zu 16 Spielern entfaltet das Spiel sein volles Potenzial. Doch dafür brauchst du einen stabilen Server.

Option 1: Enshrouded Dedicated Server selbst hosten

Seit dem Early-Access-Release stellt Keen Games einen offiziellen Dedicated Server über Steam zur Verfügung. So richtest du ihn ein:

  1. SteamCMD installieren: Lade SteamCMD herunter und installiere den Enshrouded Dedicated Server via Befehl: app_update 2857200.
  2. Server-Konfiguration: Editiere die Datei enshrouded_server.json mit deinen Wünschen: Spielername, Passwort, Spielerzahl und Schwierigkeitsgrad.
  3. Ports freigeben: Öffne die Ports 15636–15638 (UDP) in deiner Firewall und deinem Router.
  4. Server starten: Führe die enshrouded_server.exe aus – fertig! Deine Freunde können nun über deine öffentliche IP joinen.

Der Selbstbau ist kostenlos, aber nicht für jeden geeignet: Dein PC muss durchgehend laufen, und die Internet-Leitung sollte stabil sein. Alternativ mietest du einfach einen Gameserver.

Option 2: Enshrouded Server mieten (empfohlen)

Gemietete Server sind bequem, rund um die Uhr erreichbar und werden vom Anbieter gewartet. Die Einrichtung erfolgt meist per Klick im Webinterface. Hier ein Vergleich der aktuellen Anbieter:

Systemanforderungen für Enshrouded

Enshrouded ist ein Early-Access-Titel und entsprechend hungrig, was die Ressourcen angeht. Vor allem die Weltgeneration und der Shroud-Mechanismus fordern CPU und RAM. Hier die Richtwerte:

Komponente Minimal Empfohlen
CPU 2 Kerne 4 Kerne (3,0+ GHz)
RAM 4 GB 8 GB
Speicher 10 GB SSD 20 GB SSD
Internet 50 Mbit/s 100 Mbit/s

Server-Optimierung – So läuft Enshrouded flüssig

Da Enshrouded noch im Early Access ist, können Performance-Probleme auftreten. Mit diesen Tipps holst du das Beste aus deinem Server:

  • Restarts einplanen: Ein wöchentlicher Neustart beugt Memory-Leaks vor.
  • Backups aktivieren: Die Weltdateien können schnell mehrere GB groß werden. Sichere sie regelmäßig.
  • Maximale Spielerzahl anpassen: 16 Spieler sind möglich, ab 8 wird es aber oft laggy. Teste, was für deine Gruppe funktioniert.
  • Standort wählen: Ein Server in Deutschland oder Mitteleuropa sorgt für niedrige Latenz für dich und deine Freunde.

💡 Tipp: Wenn du den Server auf einem VPS hosten möchtest (z. B. Netcup oder Hetzner), installiere SteamCMD und den Dedicated Server per SSH. Achte darauf, dass der VPS mindestens 4 GB RAM und eine schnelle SSD hat. Windows VPS sind einfacher einzurichten, Linux-VPS günstiger.

Fazit

Enshrouded ist ein fantastisches Koop-Erlebnis, aber ohne einen stabilen Server wird der Spaß schnell getrübt. Für kleine Gruppen reicht der selbst gehostete Dedicated Server, sofern die Leitung mitspielt. Wer es bequem und zuverlässig mag, mietet einen Server bei einem der genannten Anbieter. Die Preise sind fair und die Einrichtung dauert nur wenige Minuten. Worauf wartest du? Starte dein Enshrouded-Abenteuer noch heute!
