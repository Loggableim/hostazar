Raft Server mieten – Dedicated Server Guide & Kosten 2026

Du liebst Raft und möchtest mit Freunden auf einem eigenen Server die endlose Ozean-Welt erkunden? In diesem Guide erfährst du alles über gemietete und selbst gehostete Raft-Server – Kosten, Setup, Mods und die besten Anbieter 2026.

Was ist Raft? Ein kurzer Überblick

Raft ist ein Open-World-Survival-Spiel des schwedischen Entwicklerstudios Redbeet Interactive, das von Axolot Games (bekannt durch Scrap Mechanic) veröffentlicht wurde. Das Spielprinzip ist simpel, aber fesselnd: Du wachst auf einem kleinem Floß mitten im Ozean auf – ohne Land in Sichtweite. Mit einer Harpune fängst du Treibgut aus dem Wasser, erweiterst dein Floß Stück für Stück und kämpfst gegen den gefährlichen Haifisch Bruce, der ständig versucht, deine Konstruktion zu zerstören.

Was als simpler Überlebenskampf beginnt, entwickelt sich schnell zu einer epischen Reise: Du schaltest Forschungsgegenstände frei, besuchst verlassene Inseln, tauchst nach Schätzen und entschlüsselst die Geheimnisse der untergegangenen Zivilisation. Der Story-Modus führt dich zu immer neuen Handlungsorten – und das alles auf deinem eigenen, ständig wachsenden Floß. Das Spiel unterstützt offiziell bis zu 8 Spieler im Koop-Modus, was es zum perfekten Gruppen-Erlebnis macht.

Warum einen dedizierten Raft-Server mieten?

Raft bietet standardmäßig einen integrierten Koop-Modus, bei dem ein Spieler als Host fungiert. Das Problem: Sobald der Host das Spiel verlässt, ist die Runde vorbei und alle anderen Spieler werden getrennt. Zudem leidet die Performance, wenn der Host-Rechner nicht ausreichend leistungsstark ist. Ein dedizierter Server löst all diese Probleme:

  • 24/7 Verfügbarkeit: Dein Raft-Server läuft rund um die Uhr – unabhängig davon, ob du selbst eingeloggt bist oder nicht. Deine Freunde können jederzeit weiterspielen.
  • Bessere Performance: Durch die Auslagerung auf einen externen Server bleibt dein eigener Rechner entlastet. Das sorgt für flüssigere FPS und weniger Lags.
  • Keine Host-Abhängigkeit: Alle Spieler haben die gleiche Verbindung zum Server – keine Vorteile oder Nachteile durch unterschiedliche Heimnetzwerke.
  • Einfaches Modding: Mit einem dedizierten Server kannst du Mods wie RaftModding installieren, um das Spielerlebnis individuell anzupassen.

Für Spieler, die keinen technischen Aufwand betreiben wollen, ist das Mieten eines Raft-Servers die ideale Lösung. Die meisten Gameserver-Anbieter haben Raft inzwischen im Programm. Die Einrichtung erfolgt meist per Klick – ohne SteamCMD, ohne manuelle Konfiguration. Hier ein Vergleich der wichtigsten Anbieter:

Option 2: Raft Server selbst hosten (VPS mit SteamCMD & Docker)

Wenn du bereits Erfahrung mit Servern hast oder auf Dauer Kosten sparen möchtest, kannst du den Raft-Dedicated-Server auch auf einem VPS (Virtual Private Server) selbst einrichten. Das erfordert Grundkenntnisse in Linux/Windows und die Bedienung über die Kommandozeile. Dafür hast du die volle Kontrolle über die Hardware und zahlst oft weniger – besonders bei Anbietern wie Hetzner, Netcup oder Contabo.

Empfohlene VPS-Konfiguration für Raft:

  • RAM: mindestens 4 GB (2 GB für Raft + 2 GB für Betriebssystem)
  • CPU: 2–4 vCores (Intel Xeon / AMD EPYC)
  • Speicher: 40 GB SSD (das Spiel + Welt-Daten)
  • Betriebssystem: Ubuntu 22.04 LTS oder Windows Server 2022
  • Bandbreite: mindestens 100 Mbit/s Uplink

Setup per SteamCMD (Linux)

  1. VPS mieten und per SSH verbinden – Empfehlung: Hetzner CX22 (ca. 3,99 € mtl.) oder Netcup VPS 1000 (ca. 5,50 € mtl.)
  2. SteamCMD installieren: sudo apt update && sudo apt install steamcmd
  3. Dedicated Server herunterladen: steamcmd +login anonymous +force_install_dir ./raft +app_update 1550160 validate +quit
  4. Raft Server starten: ./raft/RaftDedicatedServer -batchmode
  5. Ports öffnen: UDP 7654, TCP 7654, UDP 27015, TCP 27015 (Firewall konfigurieren)
  6. Autostart einrichten: systemd-Service oder Screen-Session für dauerhaften Betrieb

Setup per Docker

Noch einfacher geht es mit einem fertigen Docker-Image. Beliebte Images sind mlupica/docker-raft-server oder champii/raft-server. Ein einfaches docker compose up -d genügt – und der Server läuft:

version: '3'
services:
  raft:
    image: mlupica/docker-raft-server
    ports:
      - "7654:7654/udp"
      - "27015:27015/udp"
    environment:
      - SAVE_INTERVAL=300
      - GAME_PORT=7654
      - MAX_PLAYERS=8
      - SERVER_NAME=Mein Raft Server
    volumes:
      - ./raft-data:/app/data

Schritt-für-Schritt: Raft Server Setup (gemietet)

So richtest du einen gemieteten Raft-Server in wenigen Minuten ein:

  1. Anbieter auswählen und Server buchen – Wähle eine Konfiguration mit mindestens 2 GB RAM und 4 Spieler-Slots. Ein Server-Standort in Deutschland minimiert die Latenz.
  2. Server im Webinterface starten – Nach der Buchung erhältst du Zugang zum Webinterface. Klicke auf "Installieren" – der Raft-Server wird automatisch per SteamCMD installiert.
  3. Server-Einstellungen konfigurieren – Lege einen Server-Namen, ein Passwort (optional), die maximale Spielerzahl und die Welt-Einstellungen (Schwierigkeitsgrad) fest.
  4. Port-Forwarding prüfen – Die meisten Anbieter konfigurieren die Ports automatisch. Standardmäßig benötigt Raft die Ports UDP 7654 und UDP 27015.
  5. Server-Token nicht vergessen – Raft benötigt keinen Steam Game Server Token (anders als CS2 oder Valheim). Der Server erscheint direkt in der Server-Browser-Liste von Raft.
  6. Verbinden und losspielen – Starte Raft, klicke auf "Multiplayer" → "Server beitreten" und gib die IP-Adresse deines Servers ein.
🔧 Wichtig: Raft speichert den Spielstand automatisch in regelmäßigen Abständen (standardmäßig alle 300 Sekunden). Bei gemieteten Servern wird meist ein automatisches Backup erstellt. Bei selbst gehosteten Servern solltest du selbst für regelmäßige Backups sorgen.

Mods und Plugins – RaftModding & Erweiterungen

Raft hat eine aktive Modding-Community. Die bekannteste Plattform ist RaftModding, ein Launcher, der es dir erlaubt, Mods einfach zu installieren und zu verwalten. Auch auf dedizierten Servern kannst du Mods einsetzen – das erfordert allerdings etwas Handarbeit:

  • RaftModding Launcher: Für selbst gehostete Server kannst du den RaftModding-Launcher auf dem Server installieren. Die Mods werden dann im Mods-Ordner des Spielverzeichnisses abgelegt.
  • Beliebte Mods für Raft:
    • More Storages – Fügt zusätzliche Lagermöglichkeiten hinzu
    • Crop Growth – Beschleunigt das Pflanzenwachstum
    • Bigger Backpack – Erhöht das Inventar-Limit
    • Utility Mod – Verschiedene QoL-Verbesserungen (Items duplizieren, Teleportation)
  • Achtung: Alle Spieler, die auf einem modifizierten Server spielen möchten, müssen die exakt gleichen Mods installiert haben. Der RaftModding-Launcher synchronisiert dies automatisch.

Für gemietete Server bieten Anbieter wie Nitrado und G-Portal die Möglichkeit, Mods direkt im Webinterface zu installieren. Das ist der einfachste Weg, um Raft mit Mods zu spielen.

Performance-Tipps und RAM-Empfehlungen

Raft ist kein besonders ressourcenhungriger Server, aber es gibt einige Fallstricke. Hier die wichtigsten Performance-Tipps:

  • RAM: Für eine Runde mit 4–6 Spielern reichen 2 GB RAM völlig aus. Erst bei vielen Mods oder sehr großen Welten (hunderte Stunden Spielzeit) werden 3–4 GB empfohlen.
  • CPU: Raft ist nicht sonderlich CPU-intensiv. Ein moderner Intel Xeon oder AMD EPYC mit 2 Kernen ist ausreichend. Die Taktfrequenz ist wichtiger als die Kernanzahl.
  • Weltgröße begrenzen: Raft generiert die Welt unendlich. Je größer die erforschte Fläche, desto mehr Speicher wird benötigt. Ein regelmäßiger Neustart des Servers (alle 24–48 Stunden) beugt Speicherlecks vor.
  • Speicherlecks: Raft leidet bekanntermaßen unter Speicherlecks im Multiplayer. Ein regelmäßiger Neustart des Server-Prozesses (z. B. per Cron-Job einmal täglich) verhindert Abstürze.
  • Backup-Intervall: Stelle das Speicher-Intervall auf 300–600 Sekunden ein (via Save.Interval in der Konfiguration). Häufigere Speicherungen belasten die CPU und die SSD.
  • Netzwerk: Der Raft-Server benötigt nur ca. 1–5 Mbit/s Bandbreite. Eine 100-Mbit/s-Leitung ist also mehr als ausreichend.

💰 Unser Tipp: Wenn du Raft nur gelegentlich mit Freunden spielst (z. B. 2–3 Abende pro Woche), reicht ein gemieteter Server völlig aus. Bei täglicher Nutzung oder wenn du mehrere Gameserver betreibst, lohnt sich ein VPS.

Fazit und Empfehlung

Raft ist und bleibt eines der charmantesten Survival-Spiele, besonders im Koop-Modus. Ein eigener Server hebt das Spielerlebnis auf ein neues Level: Keine Host-Abhängigkeit, 24/7 Verfügbarkeit und die Möglichkeit, Mods zu installieren.

Unsere klare Empfehlung für 2026:

  • Einsteiger & Gelegenheitsspieler: Miete einen Server bei Nitrado oder G-Portal. ab 8–10 € im Monat bekommst du einen fertig konfigurierten Server mit DDOS-Schutz und deutscher Standortwahl.
  • Fortgeschrittene & Dauerläufer: Richte einen Raft-Server auf einem Hetzner CX22 VPS ein. Für unter 4 € im Monat bekommst du eine leistungsfähige Basis – und kannst sogar mehrere Gameserver parallel betreiben.
  • Modding-Enthusiasten: Nitrado bietet die einfachste Mod-Integration. Alternativ ist der RaftModding-Launcher auf einem selbst gemanagten VPS die flexibelste Lösung.

Worauf wartest du? Stelle deine Crew zusammen, baue dein Floß und wage dich auf die offene See – dein eigener Raft-Server macht es möglich!
