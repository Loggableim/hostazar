Einführung: Zwei Welten bei Hetzner

Hetzner Online bietet seit Jahrzehnten Hosting-Lösungen an und hat sich besonders im deutschsprachigen Raum einen exzellenten Ruf erarbeitet. Mit der Einführung der Hetzner Cloud im Jahr 2017 hat das Unternehmen den Spagat zwischen klassischem Hosting und moderner Cloud-Infrastruktur geschafft.

Während dedizierte Server seit jeher das Rückgrat des Angebots bilden, richtet sich die Cloud an Entwickler, Start-ups und Unternehmen, die flexible Ressourcen benötigen. Beide Welten teilen sich die hochwertige Infrastruktur der Hetzner-Rechenzentren in Frankfurt, Falkenstein, Helsinki und Singapur.

Die Wahl zwischen Cloud und Dedicated ist keine Frage von „besser" oder „schlechter", sondern eine Frage der individuellen Anforderungen. In diesem Artikel vergleichen wir beide Optionen systematisch und geben Ihnen das Rüstzeug für eine fundierte Entscheidung im Jahr 2026.

Was ist die Hetzner Cloud?

Die Hetzner Cloud ist eine Infrastructure-as-a-Service-Plattform (IaaS), die auf moderner Virtualisierung basiert. Kunden mieten virtuelle Maschinen (Cloud-Server), die innerhalb weniger Sekunden bereitgestellt werden können. Die Abrechnung erfolgt minutengenau – ein großer Vorteil gegenüber dedizierten Servern mit monatlicher Fixpreis-Struktur.

Das Angebot umfasst verschiedene Produkte: Cloud-Server, Cloud-Volumes (Netzwerkspeicher), Load Balancer, Floating IPs, Snapshots und ein privates Netzwerk zwischen den Instanzen. Steuerung und Konfiguration erfolgen bequem über die Cloud-Konsole im Browser, die offizielle API oder die Kommandozeile mit dem hcloud-Tool.

Besonders attraktiv ist die transparente Preisgestaltung: Schon der kleinste Cloud-Server (CX22) ist ab etwa 4,35 € pro Monat erhältlich, während größere Instanzen mit bis zu 64 vCPUs und 256 GB RAM verfügbar sind. Damit eignet sich die Hetzner Cloud sowohl für kleine Projekte als auch für produktive Unternehmensanwendungen.

Die Architektur basiert auf KVM-Virtualisierung und garantiert isolierte Ressourcen. Anders als bei vielen Mitbewerbern teilen sich Cloud-Server bei Hetzner den physischen Host nicht mit hunderten anderer Instanzen – die Performance bleibt dadurch konstant hoch.

Was ist ein Hetzner Dedizierter Server?

Ein dedizierter Server (auch Bare-Metal-Server genannt) ist eine physische Maschine, die exklusiv einem einzigen Kunden zur Verfügung steht. Bei Hetzner bedeutet das: Die gesamte CPU, der gesamte RAM, die gesamte Storage-Kapazität und die Netzwerkbandbreite gehören allein Ihnen – ohne Virtualisierungsschicht zwischen Hardware und Betriebssystem.

Das Portfolio reicht vom Einstiegsmodell AX41 mit AMD Ryzen 5 3600 bis hin zu Hochleistungsservern wie dem AX162 mit zwei AMD EPYC 9454P Prozessoren und 512 GB RAM. Für besonders speicherintensive Workloads gibt es die SX-Serie mit NVMe-SSDs oder HDD-Konfigurationen, während die EX-Serie Intel-basierte Premium-Hardware bietet.

Die Bereitstellung dauert in der Regel zwischen wenigen Minuten und mehreren Stunden, je nach Modell und Verfügbarkeit. Nach der Bereitstellung haben Sie vollen Root-Zugriff auf das System und können jedes beliebige Betriebssystem installieren – von Ubuntu, Debian und CentOS bis hin zu FreeBSD, Windows Server oder eigenen Custom-ISOs.

Die Abrechnung erfolgt monatlich per Festpreis, in vielen Fällen auch jährlich mit Rabatt. Im Vergleich zur Cloud entstehen keine Kosten für nicht genutzte Ressourcen – Sie zahlen genau die Hardware, die Ihnen dauerhaft zur Verfügung steht.

Die Tabelle macht deutlich: Die Cloud ist auf Agilität und Automatisierung ausgelegt, während dedizierte Server maximale Kontrolle und vorhersehbare Performance bieten. Welche Eigenschaft wichtiger ist, hängt vollständig vom Einsatzzweck ab.

Ein wichtiger Punkt ist die Frage der Skalierbarkeit. Während Sie in der Cloud einen Server per Mausklick von 4 auf 32 GB RAM hochstufen können, erfordert ein Upgrade beim dedizierten Server in der Regel eine Migration auf ein größeres Modell. Bei kurzfristigen Lastspitzen ist die Cloud klar im Vorteil.

Die Preisgestaltung ist bei Hetzner traditionell einer der größten Pluspunkte. Beide Produktlinien bieten ein ausgezeichnetes Preis-Leistungs-Verhältnis im Vergleich zu internationalen Hyperscalern wie AWS, Google Cloud oder Microsoft Azure. Dennoch gibt es deutliche Unterschiede in der Preisstruktur.

Was sofort auffällt: Für Workloads, die dauerhaft viel Rechenleistung benötigen, sind dedizierte Server preislich oft unschlagbar. Ein AX41 mit 64 GB RAM und dedizierten 6 Cores kostet mit etwa 51 € pro Monat weniger als eine Cloud-Instanz mit vergleichbarer RAM-Ausstattung – dafür allerdings ohne die Flexibilität der Cloud.

In der Cloud profitieren Sie von der stundengenauen Abrechnung: Ein Server, der nur für ein dreistündiges Batch-Processing benötigt wird, kostet im Cloud-Tarif nur Bruchteile eines Monatspreises. Diese Flexibilität macht die Cloud besonders für variable Workloads, Entwicklungsumgebungen und kurzfristige Projekte attraktiv.

Für dauerhaft produktive Systeme mit konstant hoher Last lohnt sich hingegen fast immer der dedizierte Server. Die fixen monatlichen Kosten ermöglichen zudem eine bessere Budgetplanung, da keine Überraschungen durch plötzlich gestiegenen Ressourcenverbrauch drohen.

Performance-Vergleich: CPU, RAM und Storage

In puncto Roh-Performance haben dedizierte Server naturgemäß die Nase vorn. Ohne Virtualisierungsschicht und mit exklusivem Zugriff auf die Hardware erreichen sie konsistent hohe Werte in Benchmarks. Allerdings hat Hetzner die Cloud in den letzten Jahren deutlich optimiert.

Die Cloud-Server der CX-Serie nutzen Shared-CPU-Kerne, die je nach Auslastung des Hosts mehr oder weniger stark konkurrieren. In der Praxis bedeutet das: Bei rechenintensiven Single-Threaded-Aufgaben kann es zu leichten Performance-Schwankungen kommen. Für die meisten Web-Workloads, Datenbanken und Container-Setups ist die Performance aber absolut ausreichend.

Die CCX-Serie der Hetzner Cloud bietet dedizierte CPU-Kerne an – ein Hybrid zwischen Cloud und Dedicated. Hier erhalten Sie die Vorteile der Cloud (API, Skalierbarkeit, minutengenaue Abrechnung) kombiniert mit der garantierten Performance dedizierter Hardware. Modelle wie der CCX63 mit 48 dedizierten vCPUs richten sich an anspruchsvolle Workloads.
