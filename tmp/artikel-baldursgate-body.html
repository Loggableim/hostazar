Baldurs Gate 3 Server mieten 2026 – D&D Multiplayer & Coop Setup

Baldur's Gate 3 hat das RPG-Genre neu definiert. Larian Studios lieferten ein episches Dungeons-and-Dragons-Abenteuer mit bis zu 4-Spieler-Koop, tiefgreifenden Entscheidungen und einer Welt die sich an deine Handlungen anpasst. Doch wie hostet man einen BG3-Server? Und welche Anbieter eignen sich für stabile D&D-Sessions?

BG3-Server-Info: Baldur's Gate 3 nutzt ein P2P-System mit einem Host. Es gibt keine offiziellen Dedicated Server von Larian. Für maximale Stabilität kannst du einen VPS als Host einsetzen.

BG3 Multiplayer – So funktioniert es

Baldur's Gate 3 unterstützt bis zu 4 Spieler im Koop. Der Host lädt die Kampagne, die anderen schreiben sich ein. Wichtig:

  • Max. 4 Spieler pro Kampagne
  • Host muss der Campaign-Besitzer sein: Nur der Host kann die Kampagne laden
  • Character Import: Spieler können eigene Charaktere importieren oder aus dem Host-Pool wählen
  • Decision-Making: Entscheidungen können von jedem Spieler getroffen werden (konfigurabel)
  • Crossplay: Steam, GOG und Mac sind kompatibel

BG3 auf einem VPS hosten – Anleitung

Wenn du einen 24/7-BG3-Server betreiben willst (z.B. für eine D&D-Gruppe):

VPS-Anforderungen

  • CPU: 2 Kerne (i5/Ryzen 5 empfohlen für Mods)
  • RAM: 8 GB (mit Mods besser 16 GB)
  • Speicher: 20 GB SSD (Spiel ist ~15 GB)
  • OS: Windows Server 2019+ oder Linux mit Steam Play (Proton)

Installation

  1. VPS mit Windows Server oder Ubuntu einrichten
  2. Steam installieren → Baldur's Gate 3 herunterladen
  3. BG3 starten → "Multiplayer" → "Kampagne hosten"
  4. Port-Forwarding: TCP/UDP 27015-27030
  5. IP und Passwort an Freunde weitergeben

BG3 Mod-Support

BG3 hat einen riesigen Mod-Support über den BG3 Mod Manager und Nexus Mods:

  • Script Extender: Für fortgeschrittene Mods erforderlich
  • Mod Fixer: Muss nach jedem Patch aktualisiert werden
  • Popular Mods: Party Size Wärter (8 Splitter), Native Camera Tweaks

Wichtig bei Mods: Alle Spieler müssen die gleichen Mods installiert haben. Der Host sollte eine Mod-Liste zum Teilen bereithalten.

BG3 Session optimieren – Performance & Settings

Netzwerk-Optimierung

  • Host sollte besten Ping haben: In BG3 ist der Host kritisch — der gesamte Gamestate läuft über ihn
  • Kabelgebundene Verbindung: WLAN kann bei BG3 zu Verzögerungen führen
  • Upload-Bandbreite: Mindestens 10 Mbit/s für 3-4 Spieler

Game Settings für Koop

  • Decision Mode: "Majority Vote" für Gruppenentscheidungen
  • Character Creation: Jeder erstellt eigenen Charakter (mehr Immersion)
  • Difficulty: Tactician für erfahrene Gruppen, Story für Casual-D&D

FAQ – Baldurs Gate 3 Server

Kann ich einen BG3 Dedicated Server hosten?

Larian bietet keine offiziellen Dedicated Server an. Du kannst einen VPS als Windows-Rechner mit Steam nutzen, der läuft und als Host fungiert.

Wie viele Spieler können BG3 spielen?

Maximal 4 Spieler im Koop. Mit dem "Party Size Wärter"-Mod bis zu 8 Spieler.

Ist BG3 Crossplay zwischen Steam und GOG?

Ja, Steam- und GOG-Spieler können zusammen spielen. Auch Mac ist kompatibel.

Brauche ich einen leistungsstarken Server für BG3?

Nein. Für P2P-Hosting reicht ein normaler VPS mit 2 Kernen und 8 GB RAM. Der Host-PC übernimmt die gesamte Spielogik.

Kann ich BG3-Mods im Multiplayer benutzen?

Ja, aber alle Spieler müssen die gleichen Mods installieren. Vor jeder Session prüfen, ob die Mod-Liste noch passt.
