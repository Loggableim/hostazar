PUBG Custom Games – So funktionieren eigene Spiele

Seit 2023 bietet PUBG offizielle Custom Game Unterstützung. Das bedeutet:

  • Private Matches bis zu 100 Spieler
  • Kartenauswahl: Erangel, Miramar, Sanhok, Taego, Paramo, Deston, Rondo
  • Custom Settings: Zone-Speed, Loot-Spawn, Fahrzeuge, Waffen
  • Tournament-Modus für organisierten Wettbewerb
  • Observer-Tool für Casting und Streams

Für Custom Games benötigst du einen API-Key von KRAFTON (dem PUBG-Publisher). Dieser kann über das PUBG-Developer Portal beantragt werden. Die Verification dauert in der Regel 3-5 Werktage.

PUBG Server selbst hosten – Anforderungen

Alternativ kannst du deinen eigenen PUBG-Server auf einem VPS oder Dedicated Server aufsetzen. Die Anforderungen sind höher als bei den meisten Spielen:

Hardware-Anforderungen (für 64 Spieler)

  • CPU: Intel Xeon E5-2680 / AMD EPYC oder besser (mindestens 4 Kerne)
  • RAM: 8 GB (empfohlen 16 GB)
  • Netzwerk: Mindestens 100 Mbit/s Upload
  • SSD: Mindestens 20 GB freier Speicher

Software-Anforderungen

  • OS: Ubuntu 20.04/22.04 LTS oder Windows Server 2019+
  • SteamCMD: Zum Herunterladen der Serverdateien
  • KRAFTON API-Key: Für Custom Game Authentifizierung

Profi-Tipp: Für 100-Spieler-Custom-Games empfehlen wir einen dedizierten Server mit mindestens 8 CPU-Kernen und 32 GB RAM. VPS-Lösungen reichen für bis zu 40 Spieler, darüber wird es instabil.

PUBG Server optimieren – Settings & Tweaks

Egal ob gemietet oder selbst gehostet — die richtige Konfiguration macht den Unterschied:

Netzwerk-Optimierung

  • Tickrate: Standard ist 30 Hz, für kompetitives Gameplay 60 Hz empfohlen (braucht mehr Bandbreite)
  • MaxPlayerNum: Nicht höher als nötig setzen — weniger Slots = bessere Performance
  • FPS-Cap: Server-FPS auf 128 oder 256 setzen

Zone-Speed anpassen

Langsame Zonen für intensive Endgame-Szenarien oder schnelle Zonen für Event-Modus. Die Zone-Speed über die LobbyData.ini anpassen.

Loot-Spawn Multiplikator

Für Custom Games können den Loot-Spawn-Multiplikator in der Konfiguration anpassen. Wer ein Battle-Royale-Erlebnis mit Spannung will, behält 1.0. Für Action-lastiges Gameplay 2.0-3.0.

FAQ – Häufige Fragen zu PUBG Servern

Kann ich einen kostenlosen PUBG Server hosten?

Nein. PUBG Dedicated Server kosten immer Geld — entweder durch Miete bei einem Hosting-Anbieter oder durch VPS/Dedicated-Server-Kosten plus ein optionaler Entwickler-Verifizierungsprozess.

Wie viele Spieler passen auf einen PUBG Server?

PUBG Battle-Royale-Spiele unterstützen bis zu 100 Spieler pro Match. Für Custom Games gelten die gleichen Limits. Event-Modi können abweichende Limits haben.

Welcher Anbieter hat die niedrigste Latenz in Deutschland?

G-Portal und Gameserver Kings haben Server in Frankfurt und bieten Pings von unter 10ms innerhalb Deutschlands. Für EU-weite Spiele sind Amsterdam und Frankfurt ideal.

Brauche ich eine Verifizierung von KRAFTON für Custom Games?

Ja, für offizielle Custom Games brauchst du einen API-Key vom PUBG Developer Portal. Der Verifizierungsprozess ist kostenlos, dauert aber 3-5 Werktage.

Kann ich PUBG-Modding auf meinem Server nutzen?

PUBG erlaubt kein traditionelles Modding wie bei anderen Spielen. Über die Custom Game Settings kannst du aber viele Parameter anpassen: Zone-Speed, Loot, Fahrzeuge, Waffen und mehr.
