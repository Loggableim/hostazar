DRG Multiplayer – So funktioniert es

Deep Rock Galactic bietet zwei Multiplayer-Modi:

  • P2P (Standard): Ein Spieler ist Host, die anderen verbinden sich. Einfach, aber Host muss online sein.
  • Dedicated Server: 24/7-Server, auf den jeder jederzeit joinen kann. Ideal für Communities.

Der Dedicated Server-Modus ist besonders für Clans und regelmäßige Gruppen interessant — der Server läuft unabhängig von einzelnen Spielern.

DRG Dedicated Server selbst hosten – Anleitung

Deep Ghost Ship Games stellt offizielle Dedicated-Server-Tools bereit. So richtest du deinen eigenen Server ein:

Hardware-Anforderungen

  • CPU: 2 Kerne (Intel i3 / AMD Ryzen 3 oder besser)
  • RAM: 4 GB (empfohlen 8 GB)
  • Speicher: 10 GB SSD
  • Netzwerk: 10 Mbit/s Upload
  • OS: Ubuntu 22.04 LTS oder Windows Server 2019+

Installation (Linux)

# SteamCMD installieren
sudo apt install steamcmd

# DRG Dedicated Server herunterladen
steamcmd +login anonymous +force_install_dir /opt/drg-server +app_update 1357090 validate +quit

# Server starten
cd /opt/drg-server
./DRGServer.sh -port=27015 -maxplayers=4

Installation (Windows)

  1. SteamCMD herunterladen und installieren
  2. steamcmd +login anonymous +force_install_dir C:\DRG-Server +app_update 1357090 validate +quit
  3. DRGServer.exe -port=27015 -maxplayers=4 ausführen
  4. Firewall-Regeln für Port 27015 (UDP) einrichten

DRG Server optimieren – Performance & Settings

Server-Konfiguration

Die wichtigsten Einstellungen in der DRGServer.ini:

  • MaxPlayers: 4 (Standard), bis zu 8 mit Mods
  • ServerName: Dein Server-Name im Browser
  • Password: Optionaler Passwortschutz
  • Difficulty: Hazard-Level 1-7 (Standard: 3)
  • MissionLength: 1-3 (kurz, mittel, lang)

Netzwerk-Optimierung

  • Tickrate: Standard 30 Hz, für kompetitives Spiel 60 Hz
  • Port-Forwarding: UDP 27015 freigeben
  • DDoS-Schutz: Bei öffentlichen Servern empfohlen

DRG Crossplay – Steam & Xbox/Windows

Deep Rock Galactic unterstützt Crossplay zwischen Steam und Xbox/Windows Store. Für Dedicated Server:

  • Server müssen im Crossplay-Modus gestartet werden (-crossplay Flag)
  • Alle Plattformen können beitreten
  • Mod-Support funktioniert nur auf PC (Steam)

FAQ – Deep Rock Galactic Server

Kann ich DRG kostenlos hosten?

Als P2P-Host ja. Für Dedicated Server brauchst du einen gemieteten Server oder VPS (ab ~5 €/Monat).

Wie viele Spieler passen auf einen DRG Server?

Offiziell maximal 4 Spieler. Mit Community-Mods bis zu 8 Spieler möglich.

Ist DRG Crossplay zwischen Steam und Xbox?

Ja, vollständiges Crossplay zwischen Steam, Xbox und Windows Store. Auf dem Dedicated Server mit -crossplay Flag aktivieren.

Welcher Anbieter ist der beste für DRG?

G-Portal bietet das beste Preis-Leistungs-Verhältnis mit One-Click-Setup und deutschen Servern.

Kann ich DRG-Mods auf dem Server installieren?

Ja, über die offizielle Mod.io-Integration. Mods werden automatisch an Clients verteilt, die dem Server beitreten.
